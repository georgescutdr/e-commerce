Online Shop - React & Node.js
This is an online shopping platform built using React for the frontend and Node.js for the backend. The application allows users to manage products, categories, promotions, and vouchers. 
It provides an intuitive interface for admins to upload products, create and assign categories, set promotions, and issue vouchers to customers.

Features

Product Management: Admins can upload products with details such as name, price, description, and image. Products can be categorized and tagged with promotions and vouchers.
Category Creation: Admins can create and manage product categories to better organize the shop.
Promotion Management: Admins can set up promotions for different products or categories. These promotions can be applied with discounts or other special offers.
Voucher Management: Admins can create and manage vouchers for customers, allowing them to apply discounts during checkout.
Product Assignment: Admins can assign products to specific categories and promotions, allowing for easy filtering and better user experience.

Tech Stack

Frontend

React: For building the user interface.
React Router: For handling navigation within the app.
Axios: For making HTTP requests to the backend.


Backend

Node.js: For building the RESTful API.
Express.js: A web framework for Node.js to handle routing and server-side logic.
MySQL: For database management, storing product, category, promotion, and voucher data.


Authentication

JWT (JSON Web Tokens): For user authentication and authorization.
bcrypt.js: For password hashing.
